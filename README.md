# Video Generator

An AI-powered web application that generates short videos from text descriptions using state-of-the-art diffusion models. Built with Gradio for an easy-to-use interface.

##  Features

- Generate videos from natural language prompts
- Customizable settings:
  - Number of frames (duration)
  - Quality steps
  - Prompt guidance strength
  - Video dimensions (256/320/384)
  - Random seed control
- Negative prompt support to exclude unwanted elements
- Simple and clean Gradio interface

## Tech Stack

- Python 3.8+
- Diffusers (Hugging Face)
- Transformers
- PyTorch
- Gradio
- MoviePy & ImageIO
